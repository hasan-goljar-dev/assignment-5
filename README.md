
  - 1️⃣ What is the difference between var, let, and const?
  - ans:-var is the older way of declaring variables in JavaScript. It is function-scoped and the same variable can be declared again.
  let is block-scoped, meaning it only works inside the { } where it is declared. Its value can be changed but it cannot be declared again in the same scope.
   const is also block-scoped, but once a value is assigned it cannot be changed later.
  - 2️⃣ What is the spread operator (...)?
  - ans:-The spread operator is used to expand or spread the elements of an array or object. It helps when copying, combining, or creating new arrays or objects.
  - 3️⃣ What is the difference between map(), filter(), and forEach()?
   ans:-forEach() runs a function on each element of an array but does not return a new array.     map() processes each element and returns a new array with the modified values.
    filter() returns a new array containing only the elements that match a given condition.
  - 4️⃣ What is an arrow function?
    ans:-An arrow function is a shorter and modern way to write functions in JavaScript, using the => syntax.
    - 5️⃣ What are template literals?
    ans:-Template literals are strings written using backticks (`). They allow variables or expressions to be inserted inside a string using ${}.
